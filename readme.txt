Please remember that slash commands sometimes take a while to register.
So if slash commands didn't appear after you type '/' , please wait patiently :D

Also, I used py-cord in this project, so you might have to remove discord.py or anything related to this, and then install py-cord (python3 -m pip instal -U py-cord --pre (--pre is very important)) so that this bot work correctly
